# docs/flow-state-parameters.md — Flow State Audio Design Reference

## Brainwave Band Targets
| State | Band | Hz Range | Binaural Beat | Carrier Hz | Generation Profile |
|---|---|---|---|---|---|
| Deep flow / focus | Gamma | 35–45 | 40 Hz | 110 Hz | Sparse, dark, stable drone |
| Alert focus | Beta | 15–30 | 20 Hz | 200 Hz | Medium, evolving, neutral texture |
| Creative / relaxed | Alpha | 8–13 | 10 Hz | 300 Hz | Dense, warm, slow modulation |
| Deep relaxation | Theta | 4–8 | 6 Hz | 100 Hz | Very sparse, airy, no transients |
| Deep sleep | Delta | 1–4 | 2 Hz | 80 Hz | Minimal, sub-bass only |

## Gamma Binaural Beat Setup (Primary for Flow)
- Left ear carrier: 110 Hz
- Right ear carrier: 150 Hz
- Perceived binaural interval: 40 Hz (Gamma)
- Source: production practice confirmed effective for focus/cognitive performance
- Use headphones — binaural beats require stereo separation

| Gamma/flow | sparse | dark | 0.15 | stable | expansive |
| Beta/focus | medium | neutral | 0.30 | evolving | room |

## Instrument Layer Guidelines
- No vocals, no speech, no lyrical content
- Preferred for flow: long sustained pads, bowed strings, soft sine sub-bass
- Preferred for beta: plucked or struck instruments with fast decay, occasional melodic fragments
- Preferred for alpha: warm pads, guitar harmonics, slow arp
- All instruments should have release time > 1.5s to avoid sharp transients

## Temporal Density Guidelines
- Gamma target: < 0.5 events/second (sparse onset density)
- Beat regularity < 0.3 (loose or arrhythmic — strict rhythm disrupts flow)
- BPM: 60–80 if any rhythmic element; avoid strong downbeats
